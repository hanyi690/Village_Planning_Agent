# 金田村 - 历史文保规划

根据金田村现状分析报告，以下是对金田村历史文化遗产保护规划的制定：

**一、规划目标**

1. 保护金田村丰富的历史文化遗产，传承客家文化。
2. 优化村庄空间布局，提升村庄人居环境。
3. 发展乡村旅游，促进村民增收致富。

**二、规划原则**

1. 保护优先原则：以保护历史文化遗产为核心，确保文化遗产的真实性和完整性。
2. 继承发展原则：在保护的基础上，合理利用历史文化遗产，促进村庄发展。
3. 可持续发展原则：实现经济发展、社会进步和生态环境保护协调发展。
4. 公众参与原则：广泛征求村民意见，提高村民参与保护的积极性。

**三、规划内容**

**（一）文化遗产保护**

1. **物质文化遗产保护**：

* **文物建筑保护**：
    * 对XX祠堂进行抢险加固，投资30万元。
    * 对老宅进行修复，投资20万元。
* **历史环境要素保护**：
    * 对炉池村古驿道进行保护和修复，投资10万元。
    * 对茶亭遗址进行保护，投资5万元。
    * 对千年古檀树和蕈树进行保护，投资2万元。
* **历史街巷保护**：
    * 保持传统街巷格局，控制建筑高度、密度，投资5万元。

2. **非物质文化遗产保护**：

* **传承人保护**：
    * 认定船灯舞代表性传承人，给予补贴和表彰。
    * 设立传习所，开展传习活动。
* **活态传承**：
    * 恢复传统节庆活动，如添灯习俗、供奉“公王”等。
    * 建立村史馆和非遗馆，展示非遗项目。
* **记录保存**：
    * 对船灯舞、传统美食、传统习俗等进行文字和影像记录。

**（二）村庄空间布局优化**

1. **村庄规划**：
    * 对村庄进行规划，明确村庄发展边界和建设限制。
    * 合理布局村庄建设用地，提高土地利用效率。
    * 优化村庄道路网络，提高交通出行效率。

2. **人居环境提升**：
    * 改善供水、排水、供电、通信等基础设施。
    * 完善公共服务设施，如教育、医疗、养老、文化等。
    * 加强村庄绿化美化，提升村庄环境质量。

**（三）乡村旅游发展**

1. **旅游资源开发**：
    * 开发古驿道、古茶亭、古树名木等旅游资源。
    * 开发船灯舞、传统美食、传统习俗等非物质文化遗产旅游产品。
    * 开发田园风光、生态农业等乡村旅游产品。

2. **旅游基础设施建设**：
    * 建设停车场、旅游厕所、旅游服务中心等旅游基础设施。
    * 优化旅游交通网络，提高旅游出行效率。

3. **旅游品牌打造**：
    * 打造“生态宜居、文化传承、旅游融合”的乡村旅游品牌。
    * 加强旅游宣传推广，提升金田村旅游知名度。

**四、保障措施**

1. **组织保障**：
    * 成立金田村历史文化遗产保护工作领导小组，负责规划实施。
    * 明确各部门职责，加强协作配合。

2. **资金保障**：
    * 积极争取上级资金支持。
    * 鼓励社会资本参与保护工作。

3. **人才保障**：
    * 加强文化遗产保护人才队伍建设。
    * 培养村民参与保护的积极性。

4. **宣传保障**：
    * 加强文化遗产保护宣传教育。
    * 提高村民保护意识。

**五、实施计划**

1. **近期（1-3年）**：
    * 完成文物建筑保护、历史环境要素保护和非物质文化遗产保护。
    * 完成村庄规划、基础设施建设和公共服务设施建设。
    * 开发旅游资源，打造旅游品牌。

2. **中期（4-6年）**：
    * 完成乡村旅游发展，实现旅游收入增长。
    * 提升村庄人居环境，提高村民生活质量。

3. **远期（7-10年）**：
    * 建设成为生态宜居、文化传承、旅游融合的美丽乡村。

**六、预期效果**

通过实施金田村历史文化遗产保护规划，将实现以下目标：

1. 有效保护金田村丰富的历史文化遗产，传承客家文化。
2. 提升村庄人居环境，提高村民生活质量。
3. 发展乡村旅游，促进村民增收致富。
4. 建设成为生态宜居、文化传承、旅游融合的美丽乡村。

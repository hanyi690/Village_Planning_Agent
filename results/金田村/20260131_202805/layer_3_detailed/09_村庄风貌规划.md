# 金田村 - 村庄风貌指引

**一、金田村风貌定位**

金田村将以“生态宜居的客家传统村落”为核心，结合“特色农业发展示范区”、“林木资源深度利用基地”、“乡村旅游目的地”以及“文化传承与旅游融合创新区”等多重定位，打造具有地域特色和时代特征的乡村风貌。

**二、风貌控制导则**

1. **风貌类型**：

   - 传统风貌型：保持原有客家建筑风格，如坡屋顶、白墙黛瓦等；
   - 现代风貌型：采用新中式、现代乡村风格，结合地域特色进行创新；
   - 过渡风貌型：对新旧建筑进行改造，实现风貌协调。

2. **建筑风貌控制**：

   - 建筑高度：核心区≤2层（7m），一般区≤3层（10m），产业区≤4层（15m）；
   - 建筑风格：传统风格、现代风格、新中式风格；
   - 建筑色彩：主色调为白色、灰色、土黄色，辅助色为木色、砖红色，点缀色为少量亮色；
   - 建筑材质：传统材料为主，适度使用现代材料；
   - 建筑构件：保留传统屋顶、门窗、阳台、马头墙等元素。

3. **立面整治指引**：

   - 保留整治：质量较好的建筑进行清洗、粉刷；有特色的建筑保留特色、局部整治；
   - 改建改造：质量较差的建筑进行立面改造；风格不协调的建筑统一风格；
   - 拆除重建：危房拆除重建；严重影响风貌的建筑拆除。

4. **景观风貌设计**：

   - 村庄入口：设置村名石、标识牌，打造入口景观；
   - 街道空间：保持建筑连续性、高度控制，采用传统材料铺装，绿化、设置街道家具；
   - 公共空间：打造文化广场、滨水空间、宅间空间等，提供休闲、娱乐场所。

5. **景观节点设计**：

   - 主要节点：村口节点、中心节点、滨水节点；
   - 次要节点：街角节点、宅旁节点；
   - 节点要素：铺装、绿化、小品、设施。

6. **视线走廊控制**：

   - 保持景观视线通透，控制建筑高度；
   - 对景设计：主要对景为山、水、标志物，框景为门洞、窗洞框景；
   - 眺望点：设置山坡、屋顶眺望点，以及观景平台。

7. **夜景照明**：

   - 照明原则：功能照明、景观照明、节能环保；
   - 照明方式：建筑照明、广场照明、绿化照明。

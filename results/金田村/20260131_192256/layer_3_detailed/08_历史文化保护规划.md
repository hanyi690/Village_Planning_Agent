# 金田村 - 历史文保规划

**一、规划背景**

金田村位于广东省梅州市平远县泗水镇，拥有丰富的历史文化资源，是客家文化的重要载体。为保护和传承金田村的历史文化，提升村庄品位，制定本规划。

**二、规划目标**

1. **保护目标**：保护金田村的历史文化遗存，维护村落风貌，传承客家文化。
2. **发展目标**：将金田村打造成生态宜居、文化繁荣、旅游兴旺的客家文化村落。

**三、规划原则**

1. **保护优先**：坚持保护为主，合理利用，传承发展。
2. **整体保护**：保护历史文化遗存、村落风貌和环境，实现整体保护。
3. **传承发展**：传承客家文化，创新发展，提升村庄品位。
4. **因地制宜**：结合金田村实际情况，制定切实可行的规划。

**四、规划内容**

**（一）文化遗产保护**

1. **物质文化遗产保护**：
    * **文物古迹保护**：
        * 对XX祠堂进行抢险加固，总投资30万元。
        * 对老宅进行修复，总投资20万元。

        * 对村落内其他具有保护价值的传统民居进行修复，总投资XX万元。
    * **历史环境要素保护**：
        * 保护金田河等水体，总投资XX万元。
        * 保护古树、古井、古道等历史环境要素，总投资XX万元。
    * **历史文化街巷保护**：
        * 保护村落内传统街巷格局，总投资XX万元。

2. **非物质文化遗产保护**：

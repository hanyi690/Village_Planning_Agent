# 金田村 - 道路交通规划

### 金田村道路交通系统详细规划

#### 1. 路网规划

**道路分级**：

- **对外联系道路**：连接X038县道和122乡道，宽度7-9米，双向两车道，设计速度30-40公里/小时。
- **村干道**：连接各自然村，宽度5-7米，双向两车道，设计速度20公里/小时。
- **村支路**：连接村民组，宽度3-5米，单车道，设计速度10公里/小时。
- **巷道**：宅间路，宽度2-3米，单车道，设计速度5公里/小时。

**路网结构**：

- 采用自由式路网结构，结合地形地貌，形成“村干道+村支路+巷道”的三级路网体系。

**道路布局**：

- 对外联系道路：沿X038县道和122乡道走向，连接金田村与外部公路。
- 村干道：沿村庄主要道路走向，连接各自然村。
- 村支路：连接村民组，形成网格状布局。
- 巷道：连接村民住宅，形成宅间路。

**断面设计**：

- 对外联系道路：采用双车道横断面，包括行车道、路肩、中央分隔带、人行道。
- 村干道：采用双车道横断面，包括行车道、路肩、人行道。
- 村支路：采用单车道横断面，包括行车道、路肩。
- 巷道：采用单车道横断面，包括行车道。

#### 2. 交通组织

**机动车交通**：

- 客货分流：对外联系道路设置客货分离车道，提高道路通行效率。
- 过境交通组织：限制过境车辆进入村庄内部，减少对村民出行的影响。

**慢行交通**：

- 步行道：沿村干道、村支路两侧设置步行道，宽度1.5米。
- 自行车道：沿村干道、村支路两侧设置自行车道，宽度1.5米。

**停车设施**：

- 公共停车场：位于村委会附近，占地面积2000平方米，设置停车位50个。
- 配建停车：住宅、公建配建停车位，比例为1:1。

**交通节点**：

- 交叉口：采用渠化交通组织，设置交通标志、标线、护栏等安全设施。
- 公交站：设置在村干道、村支路交叉口，方便村民出行。
- 出入口：设置在对外联系道路与村干道交叉口，方便村民进出。

#### 3. 道路竖向规划

**竖向设计**：

- 对外联系道路：采用平曲线设计，纵坡控制在2%-4%之间。
- 村干道：采用平曲线设计，纵坡控制在1%-3%之间。
- 村支路：采用平曲线设计，纵坡控制在0.5%-2%之间。
- 巷道：采用平曲线设计，纵坡控制在0.5%-1%之间。

**排水方向**：

- 道路路面采用雨水口收集，雨水通过排水沟、检查井等设施排入村庄排水系统。

**土方平衡**：

- 对外联系道路：填方量约5000立方米，挖方量约3000立方米。
- 村干道：填方量约3000立方米，挖方量约2000立方米。
- 村支路：填方量约1000立方米，挖方量约500立方米。
- 巷道：填方量约500立方米，挖方量约200立方米。

#### 4. 交通安全设施

**交通标志**：

- 设置警告、禁令、指示标志，引导车辆和行人安全通行。

**交通标线**：

- 设置车道线、人行横道线、停车线等，规范交通秩序。

**安全设施**：

- 设置护栏、减速带、凸面镜等，提高道路安全性。

#### 5. 道路景观

**行道树**：

- 选择乡土树种，如樟树、桂花等，种植于道路两侧，美化环境。

**绿化隔离带**：

- 设置中央分隔带、侧分带绿化，形成绿化景观。

**道路照明**：

- 设置路灯，照亮道路，方便村民夜间出行。

#### 6. 建设标准和投资

**技术标准**：

- 道路等级：对外联系道路为三级公路，村干道为四级公路，村支路为五级公路。
- 设计速度：对外联系道路30-40公里/小时，村干道20公里/小时，村支路10公里/小时。

# 金田村 - 村庄风貌指引

* 山水田园型：突出金田村丘陵山地、山水风光、植被景观的特色。

**4. 建筑材质**：

* 阳台：镂空栏杆。
* 马头墙：保留传统元素。

* 有特色的建筑：保留特色、局部整治。

**2. 改建改造**：

* 质量较差的建筑：立面改造。
* 风格不协调的建筑：统一风格。

**3. 拆除重建**：

* 危房：拆除重建。
* 严重影响风貌的建筑：拆除。

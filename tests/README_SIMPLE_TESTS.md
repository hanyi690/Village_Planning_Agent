# 简化测试文件使用说明

## 概述

三个简化版本的测试文件，用于快速生成各个子图的完整报告。

## 测试文件

### 1. 分析子图简化测试
**文件**: `test/simple_test_analysis.py`

**功能**: 生成完整的现状分析报告（10个维度并行分析）

**运行**:
```bash
python -m tests.simple_test_analysis
```

**输出**: `tests/output/analysis/analysis_simple_<timestamp>.md`

**包含维度**:
- 区位分析
- 社会经济分析
- 自然环境分析
- 土地利用分析
- 道路交通分析
- 公共服务设施分析
- 基础设施分析
- 生态绿地分析
- 建筑分析
- 历史文化分析

**预期执行时间**: ~30-60秒

---

### 2. 概念子图简化测试
**文件**: `test/simple_test_concept.py`

**功能**: 生成完整的规划思路报告

**运行**:
```bash
python -m tests.simple_test_concept
```

**输出**: `tests/output/concept/concept_simple_<timestamp>.md`

**包含内容**:
- 资源禀赋分析
- 规划定位
- 发展目标
- 规划策略

**预期执行时间**: ~30-60秒

---

### 3. 详细规划子图简化测试
**文件**: `test/simple_test_detailed_plan.py`

**功能**: 生成完整的详细规划报告（全部10个专业维度）

**运行**:
```bash
python -m tests.simple_test_detailed_plan
```

**输出**: `tests/output/detailed/detailed_simple_<timestamp>.md`

**包含维度**:
1. 产业规划
2. 村庄总体规划
3. 道路交通规划
4. 公服设施规划
5. 基础设施规划
6. 生态绿地规划
7. 防震减灾规划
8. 历史文保规划
9. 村庄风貌指引
10. 建设项目库

**预期执行时间**: ~3-5分钟

## 配置要求

确保 `.env` 文件中配置了正确的 API keys：

```env
# 使用 ZhipuAI (GLM)
ZHIPUAI_API_KEY=your_key_here
LLM_MODEL=glm-4-flash

# 或使用 DeepSeek
DEEPSEEK_API_KEY=your_key_here
LLM_MODEL=deepseek-reasoner
DEEPSEEK_API_BASE=https://api.deepseek.com/v1
```

## 特点

- ✅ **一键运行** - 无需参数，直接生成完整报告
- ✅ **自动保存** - 报告自动保存到对应目录
- ✅ **时间戳** - 文件名包含时间戳，避免覆盖
- ✅ **预览** - 控制台显示前3000字符预览
- ✅ **统计** - 显示执行时间、报告长度、完成维度
- ✅ **健壮性** - 安全的错误处理，避免崩溃

---

## 与完整测试的区别

| 特性 | 简化测试 | 完整测试 |
|------|---------|---------|
| 测试数量 | 1个（生成完整报告） | 3-5个（各种场景） |
| 输出内容 | 完整报告 | 完整报告 + 统计汇总 |
| 复杂度 | 简单 | 复杂 |
| 适用场景 | 快速生成报告 | 全面测试验证 |

## 常见问题

### Q: 报告内容为空？
**A**: 检查 API key 配置是否正确，查看错误日志。

### Q: 执行时间过长？
**A**: 详细规划子图需要生成10个维度，预期3-5分钟是正常的。

### Q: 想要只生成特定维度？
**A**: 使用完整测试文件 `test/test_detailed_plan_subgraph.py`，可以指定维度。

---

## 快速开始

1. **配置 API keys** (编辑 `.env` 文件)
2. **运行测试**:
   ```bash
   python -m tests.simple_test_analysis
   python -m tests.simple_test_concept
   python -m tests.simple_test_detailed_plan
   ```
3. **查看报告**: 在 `tests/output/` 对应目录下查看生成的 Markdown 文件

# 广东省梅州市平远县泗水镇金田村 规划细化

【产业规划】

# 一、产业发展定位

长链条、多环节、高效益的岭南特色南药产业示范基地

# 二、产业发展思路

强链——强化现有环节，加大发展力度

（统筹考虑村落现有产业的预期效益，选择最有发展潜力的南药产业作为抓手，结合村落现状，强化现有的生产种植环节，加大发展力度，实现规模化种植，让南药种得好、种得精。）

# 补链——围绕四大环节，补全南药产业链

（在加大现有生产环节发展力度后，补全上游的研发环节、中游的加工环节、下游的展销环节，围绕四大环节发展产业，形成完整的产业链条，提升产业竞争优势。）

# 拓链——发展关联产业，提升产业链复杂度

（结合村落南药产业，拓宽产业链条，发挥村落剩余产能，拓展出乡村农旅、研学教育、文创加工等关联产业，提升产业链复杂度，形成金田特色南药产业发展体系，实现经济效益最大化。）

# 三、产业体系构建

### 纵向功能维度（右侧标签）

**研发** → **生产** → **加工** → **展销**

---

### 横向产业维度（三大产业）

#### 🔹 产研基地（研发顶层）

* 南药良种培育
* 科研成果研发
* 职业农民培养
* **流向**：转化 ➔ 第一产业

#### 🔹 第一产业（生产）

* **南药种植**
  * 药材喂养 ➔ **中药养生养殖**
  * （产品流向）➔ 第二产业（南药加工工坊）
* **中药养生养殖**
  * 林下养殖 ➔ **林木种植**
  * （产品流向）➔ 第二产业（预制菜工坊）
* **林木种植**
  * （产品流向）➔ 第二产业（木材加工工坊）

#### 🔹 第二产业（加工）

* **南药加工工坊**
  * （展示）➔ 第三产业（休闲农业）
  * （产品）➔ **预制菜工坊**
* **预制菜工坊**
  * （销售）➔ 第三产业（特色农贸）
* **木材加工工坊**
  * （包装）➔ **预制菜工坊**
  * （景观）➔ 第三产业（生态旅游）

#### 🔹 第三产业（展销/三产融合）

三大板块之间互相“支持”，最终汇聚于特色农贸与终端消费场景。

* **休闲农业**
  * **细分业态**：药材采收、林鸡喂食、客家美食、养生膳食
  * **终端场景**：农事体验、田园餐饮
* **生态旅游**
  * **细分业态**：田家民宿、南药养生、森林康养、古树观光
  * **终端场景**：田园风光、森林美景
* **特色农贸（核心平台）**
  * 获取上游“支持”与“销售”汇聚
  * **核心内容**：精品南药、养生食材、养生药膳、乡村文创
  * **终端渠道**：电商销售、农夫集市

# 四、产业项目细化

# 第一产业

1. 种植特色南药：

山地种植：牛大力、岗梅、五指毛桃

平地种植：红豆杉、金钱草、草珊瑚、石参、溪黄草、金银花

2. 凭依南药养殖家禽：利用南药作为鸡饲料添加剂，提升鸡肉品质与产蛋率，预防鸡病，保障养殖效益。
3. 农业废物利用发电：通过农业废弃物资源化利用，实现绿色能源生产。
4. 南药特色农产品开发：结合南药种植与养殖，打造特色农产品品牌。

# 第二产业

1. 药材加工：组建中药加工合作社，推动各自然村对特色药材进行粗加工（如杜前溪黄草加工园）。在肩丰、园寨、杜前村建设南药精加工工坊，构建集中药交易、饮片制剂生产、仓储物流为一体的现代中药加工产业体系。
2. 加工企业溯源体系：建立药材加工溯源体系，确保产品质量与安全。

# 第三产业

# 1. 特色农贸

农夫集市：汇集村内特色产品，打造线下交易市场、特产商店和市集活动，形成金田特色农贸品牌。

IP 形象打造：设计金田特色 IP 形象，用于产品包装和纪念品，提升品牌辨识度。

乡村电商：利用网络直播、短视频带货，开设特色网店，完善物流仓储配套，降低物流成本。

# 2. 休闲农业

农事体验：在黄竹村组织游客参与南药种植、家禽养殖、农业采摘等活动。

加工体验：在肩丰村发展旅游纪念品加工产业，增强游客互动体验。

# 3. 生态旅游

田家民宿：改造营子里村民房屋，建设特色田间民宿。

南药养生：结合民宿分布，提供南药养生服务与产品。

森林康养：在自然林区设置森林驿站和森林公园，提供康养服务。

古树观光：保护古檀树林，打造观光线路和互动空间，讲好古檀故事。

# 五、产业项目布局

三心带动，五区联动。

园寨加工中心：主要发展南药与预制菜加工，同时，打造产研基地，助力成果，服务农业生产,带动金田西部发展。

黄竹旅游中心：金田村游客服务中心，主要提供餐饮、特产销售等服务。作为全
村旅游业的核心区，带动金田中部、北部发展。

肩丰加工中心：以南药加工、林业加工为核心，带动金田东部发展。

农业生产区：以合作经营的形式进行规模化的农业生产。

林下开发区：以家庭农场为形态探索种养一体的林下经济模式

休闲农业区：综合特产销售、田园餐饮、农事体验，发展休闲农业，聚集乡村人气。

康养旅游区：依托金田森林、田园自然资源禀赋，以康养为目的，结合南药文化，
发展乡村民宿，留住乡村人气。

森林保育区：十分珍惜，切实保护金田林业资源，守护好金田发展的金山银山。

# 【村庄总体规划】

# 一、空间格局

“二心二轴四区”

园寨村民中心：位于金田中南部，主要面对村民提供服务，提供村庄治理、产业发展、医疗卫生等用地。

黄竹游客中心：位于金田中东部，主要面对游客提供服务，提供文化宣传、旅游服务等用地。

旅游休闲轴：依托优质生态资源和南药产业，形成以森林康养和休闲农业为代表的旅游休闲区域。

综合发展轴：沿 122 乡道，推动形成以南药种植为中心的集农业生产、农产加工为一体的产业聚集区域，以及综合服务发展区域。

田园生活区：位于金田村中部，为村民聚居的区域，引导村庄宅基地集中建设，集约发展。

产业发展区：位于金田村东北部与西南部，布局生产加工中心，形成以南药种植为中心的产业发展区域。

青山林海区：位于金田村西部与东部，拥有丰富的林海资源，是金田村的生态基石。

旅游观光区：位于金田村中部和北部，发展集生态旅游、休闲农业的农旅发展区域。

# 三、村庄用地控制

村庄增长边界线：在尊重现有村庄自然村分布情况的基础，合理划定村庄增长边界，引导村庄集约发展。

生态保护红线：属于严格控制保护开发区域，严禁开发建设，为保障村庄基本生态安全，维护生态系统的科学性、完整性和连续性，防止村庄建设无序蔓延，在尊重村庄自然生态系统和合理环境承载力的前提下，根据有关法律、法规，结台村庄实际情况划定的生态保护红线。

永久基本农田保护控制线：永久基本农田实行永久性保护，永久基本农田的划定和管护，必须采取行政、法律、经济、技术等综合手段，加强管理，以实现永久基本农田的质量、数量、生态等全方面管护。

# 四、道路交通规划

对外交通主要通过 122 乡道连接，同时对宽度不足 8 米的原有乡道进行拓宽。

内部交通主要通过村道相连接，本次道路交通规划将村道划分为两个等级，一级村道宽度为6米，二级村道宽度为4米。同时对宽度不足6米的原有村道也进行了拓宽处理。

对于金田村原有的停车场和公交车站进行了保留和修缮。同时结合了村庄现有发展诉求和发展目标，完善了村内公共交通系统。同时，对村中的 122 乡道以及重要的村道进行保留和新增路灯设施。

依托金田村丰富的森林资源，在保护好森岭资源的前提下，根据地形的起伏，在田坑村西侧规划森林步道，营仔村作为森林步道最重要的起始服务点。森林步道通过穿越生态保护较好、原真性较好的森林区域，增加游客体验自然的机会。同时，利用金田村丰富的农田资源与水文条件，构建农田步道与滨水步道，满足不同人群对步道系统的需求。

农田步道穿越了园寨、营仔、黄竹三个主要农业生产区，并且沿途设置驿站、研学基地，让游客体验农事生产的乐趣。

滨水步道沿金田河园寨—黄竹段设立，并依据自然条件设立观景平台与亲水平台。

# 五、公共服务设施规划

基于村庄发展需求，对村庄公共服务设施进行集约化建设。在园寨，改造废弃小学，集中建设包括公共服务中心、文化服务中心、养老中心在内的村民服务综合体，与原有的村委会、卫生站等形成村庄公共服务的集聚区。在黄竹，结合地理区位，建设游客服务中心。

同时，基于公共服务的普惠性，考虑各自然村的现状，适当补充了一些篮球场、

健身设施和垃圾收集点，更好满足村民日常生活的需要。

# 六、基础设施规划

拟在黄竹、肩丰等村增设污水处理池，全村范围铺设污水处理管道，在肩丰、炉池、田坑增设公共厕所。通过增设服务覆盖全村的基础服务设施，改善村落基础服务设施缺乏且分布不均的现状，从而提升村民居住体验，切实为村民生活带来便利。

# 七、生态绿地规划

根据实际情况，拟在每个自然村增设不同规模的村民活动场地。在建设用地范围内增设公园绿地和活动广场，在范围外划定森林公园区、生态保育绿地等区域，根据各自定位采取不同的保护措施。最终打造以村域整体保护为前提，以山林保护为主体，以村落风貌提升为抓手的生态宜居、绿色和谐的金田村绿地系统和村庄生态格局。

水域方面的生态修复保护措施主要为砌河护坡、河道绿化与河底整治，计划在公王河黄竹、田坑、园寨河段开展生态整治工作，整治面积约54公顷。通过土石堆砌达到巩固河堤的效果，考虑采用生态河岸、种植乡土植物等形式绿化河边人行道。通过河底清淤疏通，提升河道水质。由此重振金田村水生态。

将生态保护红线范围内的绿地及其外围一定范围的绿地规划成生态保育绿地区，区内多为林地，以保护、恢复金田村自然生态为主要功能。生态保育区内禁止开发活动，减少人类活动干预，以形成天然绿色的生态环境。

依托古檀林资源，坚持保护优先原则，将营子里部分林地划定为古檀林主题森林公园。充分利用与盘活林地和林下资源。通过规划林下观光步道、设置标识牌等必要设施等举措，满足游客观光、游乐等旅游活动要求，给游客以亲近自然、领略古檀之美的旅游体验。

除以上区域之外的林地，均划为一般林地。一般林地区面积较大，在日常生活中应坚持保护优先原则，不随意砍伐林木。营造生态绿色、自然和谐的林地格局，实现人与自然和谐相处，生态资源得到良好保护。

耕地保护：坚决遏制耕地“非农化”现象，对耕地实施严格的管控：严禁违规占用耕地进行绿化造林、超标准建设绿色通道、挖湖造景、从事非农建设、自建农房等行为。对于确有需要，不可避免占用耕地的项目，应经过专家论证，且严禁占用永久基本农田。深入推进耕地污染防治，采取严禁向耕地排放生活生产污水和垃圾，控制农药化肥使用，及时回收农膜等措施，避免在生产生活中对耕地土壤造成污染。同时，实行耕地轮作休耕制度，保护耕地地力。

根据实际情况，在建设用地范围内增设公园绿地、文化广场、活动场地。每个自然村增设不同规模、不同类型的村民活动场地。如在林木资源丰富的营子里建设面积超千平的古檀公园，在用地较为紧张的清明建设小范围活动场地。充分考虑建设可行性，竭力满足村民活动健体的日常需求。

# 八、防震减灾规划

保留利用原有的 3 处坑塘，将其作为调蓄池，作为储水空间。通过 GIS 软件进行水位上涨模拟，在村庄的低点规划设计了 3 处调蓄池，减缓村庄被洪水淹没。在村庄集聚区规划设计水渠，加快雨水的排放。完善河道整治措施，保证河道通畅。

规划 122 乡道为主要的逃生和消防救援道路，村道为次要的逃生和消防救援道路。将规划后拥有 8 处广场和 2 处村名公园作为抗震设施。黄竹作为 122 乡道金田段的中部区位，在此设立村庄微型消防站有利于村中的消防救援。选取防洪排涝规划中靠近自然村的调蓄池作为消防的蓄水池，为消防救援提供易得的水资源。

# 【历史文化保护规划】

# 一、保护思路

深入挖掘乡村历史文化资源，提出历史文化景观整体保护措施，明确管制规则。

保护历史建筑与文化的前提下提出整合串联沿线历史文化遗产、生态、旅游等资源的策略。

# 二、保护活化利用规划

对于金田村内的祠堂建筑应结合实际需要加以修缮。在修缮过程中应维护其原有的主体结构外部风貌不变的前提下，进行维护加固，并统一各时期祠堂的外貌风格和空间尺度、形态等方面设计。对于祠堂内部的活化，一方面，修缮祠堂满足村内的祭祀等活动需求；另一方面，将闲置的祠堂修缮活化利用，修建公共文化设施如小书屋、自习室、村民文化活动中心以及优秀传统文化价值和现代核心价值宣传中心。

对于金田村的千年古檀树和古檀林，一方面要加以妥善保护；对村内部的古树名木挂牌保护，并定期检查，在发现病虫害等情况后及时联系专业人员救治等。另一方面要规划以树冠投影向外5米为保护范围，结合原有的历史景观资源特征，突出原汁原味的乡土特色。在保护的基础上将古檀林设计为主题公园，介绍古檀与金田的前世今生，作为金田历史文化建设的一部分。

对于金田村的传统文化习俗，可推举村内有威望的乡贤作为添灯仪式的主持人，鼓励村内手艺人申请非遗专利，作为非遗传承人等，保护原生文化底蕴。一方面，改造废弃、闲置的祠堂成村内文化展示基地，在祠堂内举行添灯等具有金田特色的文化活动，彰显文化底蕴与特色。另一方面，将传统文化与现代元素相融合，策划举办如春日仙人板文化节、添灯直播文化节等主题活动，吸引更多游客了解金田特色文化并参与其中，活化村内非物质文化。

# 【村庄风貌指引】

# 一、景观风貌指引

耕地景观：禁止占用基本农田，减少对其他农田的占用。通过种植不同种类的作物增加农田多样性；采取轮耕、间种等耕种手法种植农作物；提升农业生产设施美学价值。根据时令在各区内的农田地块种植不同的药材，并补充种植观赏性较强的农作物。农民在自然环境等影响下种植农作物，修建景观小品，形成简易完整却富有层次感的生产面。

林地景观：在保护好生态环境的前提下依形就势有序安排种植山林，结合实际按季相种植乡土植物，在不影响经济型的前提下增强山林可观赏。顺应山地形势肌理，根据高程、海拔气温变化种植适宜气候的植株，丰富景观层次。在林区种植对比较为明显但与总体环境协调的植物如罗汉松、台湾相思、杉木等，增强可观赏性。结合实

际需要发展林下经济。

房前屋后景观：宅旁和庭院绿化因充分尊重村民的意愿喜好。绿化植物必须具有易管理性和实用性，种植种类须采用乡土树种和攀缘植物，景观需保持一定的整体性。禁止房前屋后废弃杂物和垃圾的堆放，禁止随意粘贴广告或涂鸦，维持宅旁整洁。适当种植经济作物，发展庭院经济。庭院围墙的外墙可以用攀缘植物进行覆盖美化点缀。公共建筑可考虑种植观赏性植物进行美化。可根据村民发展意愿，结合村民的经济情况，适度发展屋顶绿化。花基、树池等物品需用富含乡土特色的材料进行建设。合理摆放农业工具杂物和家具小品进一步展现金田村的农家氛围。

河流景观：村中河流景观绿化需尽量保留、利用现有的自然形态，以生态护坡的方式整治边坡和岸线。在景观较好的河流空间应设置亲水景观平台，挖掘景观的生态服务价值。水深低于1.5米的水体宜种植挺水植物，水深高于1.5米的水体宜种植沉水植物，充分利用植物的净化作用和景观美化作用。村中河流绿化可采用村庄现有传统农业植物，选用经济树种，增加一定的经济收益。

坑塘景观：保护村中现有的坑塘，改善坑塘的水质，维持水流的汇入，保证坑塘的蓄水量。部分坑塘可以设置如荷花的水生植物，增加坑塘的净化能力，增加池塘的绿化效果。在池塘周围布置现有的乡土植物，保护坑塘的环境；在坑塘边坡采用生态驳岸设计，铺设保持水土的植物；在池塘中央设置荷花等观赏植物，增强景观效果。沿坑塘建设滨水步道或滨水公园，为人们提供休闲环境。

村庄入口景观：坚持整体出发原则，设计从金田村整体形象出发，彰显金田村特色，展现金田村魅力；坚持因地制宜原则，选用乡土材质，如瓦片、文化石、青砖、夯土等进行创意重构，融入乡村环境；坚持以人为本原则，入口景观小品和设施的空间布局应满足人体尺度和人的行为需求。与村落内部保持一定距离，保证村落内安静有序的生活环境；打造不同层次的入口景观，移步换景，与交通环境相匹配。

“四小园”：对村内房前屋后的小菜园、小果园、小花园、小游园进行改造，根据不同村落的定位方向打造不同形式的“四小园”风貌。以小微尺度建设“口袋公园”、小微绿地、美丽菜园；选用具有乡村特色的建材，如青砖、竹条等；增设相关基础设施和服务设施。

祠堂：对传统祠堂进行加固、外观整饬，保持和恢复传统风貌和特色，并赋予其新的功用。祠堂可活化为农家书屋、乡村历史文化展览馆、村民活动中心以及传统作坊等。充分尊重现状，保留祠堂现有古韵，在此基础上再进行功能赋予和场地活化。

村民活动交流场地：生态化建设，硬质铺装与生态透水铺装相结合，打造自然和谐生态化的植物景观；人本化建设，增设桌凳、健身、环卫等设施，符合人体工学。提高村民活动交流时的体验；集约化建设，打造集健体、交流、日常活动、晾晒等功能于一体的多功能活动场地，提高使用频率。优先选择粤北乡土植物进行种植，如紫薇、桂花等；在建设时应选择合适的铺装，注意防滑、儿童友好等要求。

乡道：基本实现乡道路面硬底化，逐步推进乡村道路面黑底化建设。根据道路原有的景观情况，宜保留和美化景观性较好的道路绿化的同时，采取清除杂草，裸地复绿，补植行道树等方式，优化道路景观；对于那些景观资源较好的路段，宜留出视觉通廊，与周围环境相呼应。修整山体切坡，建议采用植物型护坡与硬质型护坡相结合的方式进行护坡处理。对于道路的滨水空间，建议采用硬质化的驳岸处理方式，加配防护栏，保障行车安全。

村道：逐步实现村道硬底化。对原有植物景观不佳或土地裸露的路段进行绿化整治。位于村庄生活空间内的村道，可以打造集乡风、乡韵为一体的景观小品，提升道路景观的趣味性。建议结合金田南药种植特色，路旁种植既具景观性又易打理的南药

草本，如：野菊花、蒲公英、溪黄草等。

巷道：巷道路面应因地制宜采用块石、预制混凝土方砖或当地传统铺装（如：麻石板、青砖）等进行硬化，体现乡土特色。建议利用巷道路旁的空置地块，打造小公园、小花园、小菜园，美化巷道景观的同时形成街巷的休闲交往空间。

步道：田间步道结合步道生产功能和路旁农田大地景观，打造既具景观观赏性又具实用性的田间步道。以森林康养为主题打造特色森林步道，建议采用既有康养功能，又具景观观赏功能的乡土树种（如：香樟、悬铃木、竹等）进行沿路绿化布置；同时，在适应处设置座椅、休息亭等以供人休憩。
